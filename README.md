# Qt-uEye
Display the images of the IDS cameras by uEye and OpenCV in Qt

## Compile the Project
Install Qt (4.15.2)
    https://www.qt.io/

Install OpenCV (4.2)
    https://opencv.org/

Install IDS Software Suit (4.94)
    https://www.ids-imaging.us/home.html
    
Compile the project with Qt and Qt Creator.
